В этой системе используются только две цифры - 0 и 1. Используется в вычислительной технике.

В восмеричную систему счисления:
   Сначала необходимо разделить двоичное число на группы по 3 цифры начиная справа, добавляя нули, если нужно. Далее каждую группу следует заменить соответствующим восьмеричным символом. Например:
   Двоичное число: 11010010
   Разделяем на группы: 001 101 001 0
   Заменяем на восьмеричные символы: 3 5 1 0
   Таким образом, двоичное число 11010010 будет равно 3510 в восьмеричной системе счисления.

В десятичную систему счисления:
   Чтобы выполнить перевод, нужно умножить каждую цифру двоичного числа на соответствующую степень числа 2 и затем сложить полученные результаты. Например:
   Двоичное число: 11010010
   Умножаем каждую цифру на степень числа 2: (1*2^7) + (1*2^6) + (0*2^5) + (1*2^4) + (0*2^3) + (0*2^2) + (1*2^1) + (0*2^0)
   Складываем результаты: 128 + 64 + 16 + 2 = 210
   Таким образом, двоичное число 11010010 будет равно 21010 в десятичной системе счисления.

В шестнадцатеричную систему счисления:
   Аналогично переводу в восмеричную систему, нужно разделить двоичное число на группы по 4 цифры начиная справа, дополняя нулями при необходимости. Затем каждую группу следует заменить соответствующим шестнадцатеричным символом. Например:
   Двоичное число: 11010010
   Разделяем на группы: 0011 0100 10
   Заменяем на шестнадцатеричные символы: 34 2
   Таким образом, двоичное число 11010010 будет равно 342 в шестнадцатеричной системе счисления.
